### Installation

```shell
brew tap mongodb/brew
brew update
brew install mongodb-community@8.0
brew services start mongodb-community@8.0
mongosh
```

### Create Collection

```
use blog
db.createCollection("posts")
```

### Insert One

```
db.posts.insertOne({ id: 1, title: 'blog 1' })
```

### Get All Data

```
db.posts.find({})
```

### Get Filtered Data

```
db.posts.find({slug:'blog-1'})
```

